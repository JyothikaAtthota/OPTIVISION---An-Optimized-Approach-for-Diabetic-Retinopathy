About Optivision

Optimized Approach for Diabetic Retinopathy Detection

Project Abstract

Diabetic Retinopathy (DR) is a major cause of preventable blindness caused by diabetes-related retinal damage. Optivision is an AI-based system for real-time DR detection using Deep Learning and Computer Vision in PyTorch. The system uses a CNN model with transfer learning (ResNet50) trained on retinal fundus images to classify DR into multiple stages — from No DR to Proliferative DR.


The automated tool enhances diagnostic speed, assists ophthalmologists in large-scale screening, and supports early intervention — making eye care more accessible, especially in rural and resource-limited regions.
